GLD Motorsport — Setup Guide

Get on track
in minutes.

Five quick steps and you're racing on the GLD server. We'll check each one off as you go.

Step 1 of 5

Join the Telegram Groups

Every car, track, and pack on the GLD server is completely free — no purchases, no subscriptions. Telegram is how we distribute everything. We know it's unconventional. Just trust it. 😉

Join both groups — Assetto Corsa Collection is the main library, Assetto Pirate covers the rest.

💡Once you're in, you'll have access to thousands of cars, tracks, and mods — all free.
Assetto Corsa Collection Assetto Pirate
Step 2 of 5

Install Content Manager

Replaces Assetto Corsa's stock launcher with a far better interface. Handles server browsing, settings, and mod installs. Cars, tracks, car packs, and PP filters all install via drag and drop into CM.

✓Content Manager is safe to use for installing cars, tracks, car packs, and PP filters.
Download Content Manager
Step 3 of 5

Install CSP & Pure — Manually

Custom Shaders Pack (CSP) unlocks the advanced graphics and physics features most mods depend on. Pure overhauls weather, sky, and lighting — it works alongside CSP to make the sim look and feel completely different.

⚠️Do not install CSP or Pure via Content Manager. Both must be installed manually — the tutorial below walks you through both at once.
CSP & Pure — Telegram Install Tutorial
Step 4 of 5

Install SOL — Manually

SOL handles the day/night cycle, sun positioning, and environmental lighting. It completes the visual stack alongside CSP and Pure.

⚠️Do not install SOL via Content Manager. Install manually — just extract the folder into your Assetto Corsa directory. Very straightforward.
Download SOL
Step 5 of 5

You're almost on track.

That's everything. Click below to connect directly to the GLD Motorsport server — make sure Assetto Corsa is already running first.

🏁The link below opens Content Manager and drops you straight into the server. No IP hunting required.
Join GLD Motorsport Server
🏁

You're ready to race.

Everything is set up. Need to join the server? Click below — make sure Assetto Corsa is already running.

Join GLD Motorsport Server
